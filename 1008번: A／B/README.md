## 문제

[문제 링크](https://boj.kr/1008)

## 입력

첫째 줄에 A와 B가 주어진다. (0 &lt; A, B &lt; 10)

## 출력

첫째 줄에 A/B를 출력한다.&nbsp;실제 정답과 출력값의 절대오차 또는 상대오차가 10<sup>-9</sup> 이하이면 정답이다.
